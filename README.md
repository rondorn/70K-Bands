# Open Metal Fest Suite

Open Metal Fest Suite is an open-source platform for building mobile applications for metal festivals and music events.

The project provides a proven framework for festival discovery, scheduling, notifications, rankings, artist information, and attendee engagement across both iOS and Android.

The suite was originally developed to power the 70K Bands application for the 70,000 Tons of Metal cruise and has evolved into a reusable platform that can support multiple festivals.

---

## Current Implementations

### 70K Bands

The unofficial companion application for the 70,000 Tons of Metal cruise.

Features include:

- Band discovery
- Personal rankings
- Schedule planning
- Show reminders
- Community statistics
- Offline access

### MDF Bands

The unofficial companion application for Maryland Deathfest.

Built on the same Open Metal Fest Suite platform while providing festival-specific content and configuration.

---

## Project Goals

Open Metal Fest Suite exists to:

- Provide festival attendees with better planning tools
- Reduce development effort for festival organizers
- Encourage community-driven content creation
- Keep festival software open and portable
- Enable multiple festivals to share a common technology platform

## Core Features

### Band Discovery

Browse artists, genres, logos, and external resources.

### Personal Rankings

- Probably Not
- Won't See

- New band announcements

## Technology

### Mobile Clients

- iOS (Swift)
- Android (Java)

### Design Philosophy

Festival-specific data should be configurable whenever possible.

The platform is designed so that new festivals can be added through configuration and content management rather than requiring extensive application development.

Most festival-specific behavior is driven by data files and configuration rather than application code.

## Project History

The Open Metal Fest Suite platform began as the 70K Bands application for the 70,000 Tons of Metal cruise and has been continuously maintained and enhanced for over a decade.

Over time, the platform has expanded beyond a single event and now serves as the foundation for multiple festival applications while remaining free and open source.
